About OptiCrop Engine

An analytical breakdown of the system, models, and features.

Problem Statement

Traditional farming methods heavily rely on historical averages and trial-and-error, which are increasingly unreliable due to climate change and volatile weather patterns. Planting the wrong crop leads to poor soil health, wasted fertilizer investments, water pollution, and low yield. Crop optimization is critical for sustainable food production and farmer livelihoods.

Project Objectives

  • Collect chemical and weather metrics to make data-driven crop recommendations.
  • Perform automated training of 6 different classification models to find the highest accuracy.
  • Deliver fertilizer advice and farming instructions to promote healthy crop yields.
  • Render interactive performance charts (feature importance, confusion matrix).

Machine Learning Classifiers

We compare six classic machine learning classification algorithms to select the best predictor:

1. Logistic Regression

Models class probabilities using a logistic function. Serves as a solid linear baseline.

2. Decision Tree

Non-linear model splits dataset using feature information gain. Highly interpretable.

3. Random Forest

Ensemble of decision trees. Reduces overfitting through bootstrap aggregating (bagging).

4. K-Nearest Neighbors (KNN)

Instance-based learning classifier. Recommends crop based on proximity to historical soil profiles.

5. Support Vector Machine (SVM)

Projects input features into higher dimensions using kernels to locate optimal decision boundaries.

6. Gaussian Naive Bayes

Probabilistic classifier based on Bayes' theorem assuming feature independence. Computationally fast.

Dataset Description

The model trains on the standard Crop Recommendation Dataset, which includes 2,200 rows covering 22 crop classes:

Rice Maize Chickpea Kidneybeans Pigeonpeas Mothbeans Mungbean Blackgram Lentil Pomegranate Banana Mango Grapes Watermelon Muskmelon Apple Orange Papaya Coconut Cotton Jute Coffee

Tech Stack

Built using standard modern developer tooling: Python 3.11, Flask, Scikit-learn, Pandas, NumPy, Joblib, Matplotlib, Seaborn, HTML5, Vanilla CSS3, Bootstrap 5, and JavaScript.
